# Northpoint Digital

> Premium websites for Hamilton businesses — and remotely across Canada.

A web studio building fast, modern websites for small businesses. Monthly hosting + maintenance plans, no surprises.

🌐 **Live site:** https://northpointdigital.work

---

## Tech

- [Astro 5](https://astro.build) — static site generator
- [Tailwind CSS v4](https://tailwindcss.com) — utility-first styling
- Deployed on [Vercel](https://vercel.com)
- Domain + DNS on [Cloudflare](https://cloudflare.com)

## Local development

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # production build to ./dist
npm run preview  # preview built site locally
```

## Deployment

Pushes to `main` auto-deploy via Vercel.
